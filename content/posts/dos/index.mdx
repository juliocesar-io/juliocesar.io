_A calm park, New York_

![NYC](./dos-water.jpeg)

Pienso en tu amplio mirar, <br/>
incesante, casi _dilettante_. <br/>

El tiempo, <br/>
hasta ahora inmutable, precisa tus reglas para esquivar. <br/>

Vida calculable, <br/>
ausente de novedad. <br/>

El tiempo, <br/>
astuto como es, <br/>
medir _ha_ de olvidar. <br/>

Vacilante, entre dos, <br/>
dos almas, <br/>
dos minutos, <br/>
dos segundos. <br/>
¿Dos tiempos? <br/>

¿Escuchas esta queja? <br/>

Como cascada, <br/>
inevitable y fluvial. <br/>
No quisiera con tus reglas navegar. <br/>

Desvía el tiempo su caudal. <br/>
Estético, atractivo y funcional, <br/>
también con dulce y misteriosa densidad. <br/>

Arquitecto de aromas, <br/>
lo puedo nombrar. <br/>
Quejas no podrá escuchar, <br/>
solo aromas recordar. <br/>

Registrados cuidadosamente, <br/>
en pensamientos y más allá, <br/>
donde el suelo solo sentirás girar. <br/>

Fugitivo e impune, <br/>
de filo amplio, <br/>
cortante de la continuidad. <br/>

¡Justicia he de implorar!, <br/>
¡Por los tiempos ociosos que se me van! <br/>
