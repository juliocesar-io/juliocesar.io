A priori perfumas la cima <br/>
Instauras quehaceres <br/>
Porque de los mediadores eres  <br/>
Sublimas rutina casi sonando como ruina <br/>

Como evanescente procedes  <br/>
Entre lo intrascendente, urgente y biológico <br/>
Bajando cimas, navegando mares <br/>
Saboreas el infinito <br/>
En un gramo de sal en tus labios <br/>
Un producto de la misma gracia <br/>
Del continuo enredo <br/>
Fractal y equivalente <br/>
De la vida y la muerte <br/>

Del sufrimiento haces arte <br/>
Del descubrimiento progreso <br/> 
De los mares perfumes <br/>
y de las las cimas saberes <br/>
Cuentas estrellas pero alcanzas mas placeres <br/>

Piensas en la tumba <br/>
Lo placentero de ser perecedero <br/>
En el absurdo del suicidio <br/>
No encontrando placer ulterior <br/>
Inclusive un ser superior <br/>
Siendo el destino nada más que salino <br/>

Unos labios no tocados <br/>
Besos escritos no entregados <br/>
Vida no gastada <br/>
Bajo noches estrelladas <br/>
En la lentitud de las miradas <br/>
Una totalidad fragmentada <br/>

Infalible lo que ves <br/>
Imposible ver al sol perenne <br/>
Sin la impotente ceguera cósmica   <br/>
Posible soñar y desear  <br/>
Y es todo un placer  <br/>
Uno realmente ulterior  <br/>
Proceder, ser y perecer  <br/>
En nada más que polvo estelar  <br/>
