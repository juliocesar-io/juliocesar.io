¿Nos esperarán las estrellas?  <br/>
Mientras podría soñar  <br/>
Soñar despierto  <br/>
Seducir al tiempo  <br/>
 
O mejor aun  <br/>
Ganar el tiempo   <br/>
Con etéreo recuerdo  <br/>
y sueño casi eterno  <br/>
¿Son los sueños recuerdos o deseos?  <br/>
¿Seré solo yo y mis circunstancias?  <br/>
 
Entre las montañas y el océano   <br/>
El sol centella en tus ojos  <br/>
Con la luz instaurada en tu mirada   <br/>
¡De solo verte bailar!  <br/>
Bailar en lo etéreo   <br/>
Sin prisa ni pausa como el firmamento  <br/>
 
Yo en mi primor a lo efímero  <br/>
Mil imágenes, mil caminos  <br/>
Un camino que conduce a tus brazos  <br/>
Una imagen de tus labios  <br/>
Sonríes, tierna como eres  <br/>
Las estrellas, ya ajenas   <br/>
Pues cesan ellas; mientras tu sonríes   <br/>
Mil de esto, mil de aquello  <br/>
Cierne todo lo bello  <br/>
 
¿Encontraré una galaxia de lo tierno en mil quinientos desvelos?  <br/>
Tendría que cartografiar  <br/>
Pero sabes que elegiría cantar  <br/>
Recitar hasta que el sol no suba ni baje  <br/>
Pues feliz el sol estará; en su eterno dormitar  <br/>
¿Y las estrellas?   <br/>
– Claras y brillantes  <br/>
Como la vez desde acá  <br/>
¡Imagínate allá!
