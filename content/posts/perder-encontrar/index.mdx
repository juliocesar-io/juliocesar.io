_Cundinamarca, Guatavita_

![Cundinamarca, Guatavita](./perder-encontrar.jpg)

_Encuentro:_ <br/>

Tus ojos, <br/>
La simetria, <br/>
un instante, <br/>
una pausa. <br/>

_Pierdo:_ <br/>

Un lugar, <br/>
fuera de alcance, <br/>
libre del tiempo, <br/>
continuo, abundante. <br/>

Pierdo el suelo, <br/>
y lo encuentro. <br/>
Lo vuelvo a perder, <br/>
y lo vuelvo a encontrar <br/>
¿Así te sientes querida rueda? <br/>

De sujeción nula, <br/>
La verdad pura y desnuda. <br/>

Absurda alquimia, <br/>
incompresible probabilidad, <br/>
vago relieve, <br/>
inasequible realidad. <br/>

Solo a través de un cristal, <br/>
Poseo este cuerpo, <br/>
transparente, aislado, <br/>
vertido, asombrado. <br/>

Del _panóptico_, fugitivo, <br/>
siempre alegre de perderse, <br/>
en el mundo de las cosas. <br/>
en las _cosas de los mundos_. <br/>
Material o inmaterial, <br/>
inteligible o sensible. <br/>

He de transitar un puente, <br/>
flotando vaporoso, <br/>
pensando deseoso, <br/>
Insistente o valiente. <br/>

Cuan intrascendente, <br/>
diáfana de luz, <br/>
te canto _modernidad_, <br/>
Transparente, frágil de cristal, <br/>
fría y dura de metal. <br/>

¿Me revocas este deseo? <br/>
Tan intenso placer de vivir <br/>
¿Dónde estás tenuidad? <br/>
¿Con quien estas continuidad? <br/>

Experta en aplazamiento, <br/>
¡Olvida ya la cantidad ! <br/>
¡Lo infinito – es _cualidad_ ! <br/>

Ven, te entrego este cantar, <br/>
_tócalo_: es palpable, <br/>
como el tiempo: consonante, <br/>
como un _armoa_: fulgurante. <br/>

Lo perderás, <br/>
lo encontrarás, <br/>
lo tienes, <br/>
tenlo un poco más, <br/>
ten, toma mis labios, <br/>
besa sin perder, <br/>
mira sin encontrar. <br/>

Recuerda, _es la vida un carnaval_. <br/>
